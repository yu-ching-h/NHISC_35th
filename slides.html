課程資源 & 技術專題

[SYSTEM INFO] 這裡記錄了 NHISC 本學期的基礎課程教材，以及特邀大專院校資安教授親授的高階資安攻防專題。

特邀專題：大專院校教授資安實戰班

【專題一】Web 安全與 WAF 繞過實務

核心技術：302 跳轉觀測、前端驗證不安全性、路徑走訪 (Path Traversal)

深入探討 Web 漏洞原理。教授帶領我們使用工具截獲 302 Redirection 跳轉前的原始封包以獲取敏感資訊；並解構後端 WAF 過濾機制（如舊版 PHP `parse_url` 解析錯誤導致的雙斜線漏洞），實作目錄跳轉（`../`）存取伺服器敏感檔案。

【專題二】JS 邏輯解密與 AES 現代密碼學

核心技術：混淆腳本逆向、CryptoJS 應用、金鑰提取

學習如何分析被混淆或加密的前端 JavaScript 驗證腳本。教授指導我們利用數學公式從邏輯中反推解密演算法，並實務提取網頁中的 Hex 格式金鑰 (Key) 與初始向量 (IV)，結合 CyberChef 工具進行現代對稱式加密 AES 的解密實戰。

【專題三】Windows PE 結構與二進位逆向

核心技術：PE 結構分析、十六進位檢視、惡意函數動態/靜態取證

邁入底層逆向工程！教授詳細解構 Windows 執行檔的 PE (Portable Executable) 結構，包含 DOS Header、Section Table (.text, .data) 等。並手把手教學使用 HxD 進行十六進位編碼分析、PE Studio 靜態分析導入函數 (Imports) 以及 Resource Hacker。

【專題四】數位鑑識：進階隱寫術取證

核心技術：Magic Number 分析、檔案尾端附加數據、音訊頻譜分析

實戰 CTF 常見的數位取證技術。教授帶我們認識各種檔案格式的魔術字元（Magic Number），並利用 binwalk 地毯式搜索並分離出隱藏在圖片後方的壓縮檔；同時進階涉足音訊隱寫，使用 Audacity 觀察頻譜圖來解析撥號音 (DTMF) 並還原出 Flag 字串。
